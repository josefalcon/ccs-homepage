  • Flaco lock service
    The Flaco lock service is a RESTful service for creating and managing distributed locks across an HTTP connection. Using basic HTTP requests, users manipulate distributed locks on arbitrary strings. Flaco supports asynchronous requests through a callback mechanism. More information on Flaco can be found on google code. Due to legal issues, the source code is unavailable at this time.
